## 1. mart Campus 社團資訊系統
* **包含資料**：`Code`程式原始碼、`系統介紹` (連結)。
* **專案內容**：針對校園社團資訊分散的問題，設計一套整合資訊管理與追蹤功能的系統，增進社團幹部及想參加社團之學生的方便。
* **實作課堂**：程式設計
* **負責部分**：專案規劃 (WBS)、部分 Java 程式碼撰寫、程式介面與運作邏輯設計。

## 2. Let's Beat Google 網頁搜尋
* **包含資料**：`Code`網頁原始碼、`書面報告` (系統完整說明)。
* **專案內容**：實踐資料結構原理，透過層狀結構與排序演算法，模擬搜尋引擎根據關鍵字出現次數進行結果排序的底層邏輯。
* **實作課堂**：資料結構
* **負責部分**：WBS 進度規畫、部分排序邏輯代碼撰寫，並運用 AI 工具進行效能優化與邏輯偵錯。

## 3. JELLYCAT 玩偶真偽辨識設計
* **包含資料**：`系統構想計畫` (網頁構想完整計畫)。
* **專案內容**：模擬商品驗證流程，設計一套自動化辨識系統模型，深入探討實務運作中可能遇到的邏輯衝突與驗證挑戰。
* **實作課堂**：系統分析與設計
* **負責部分**：繪製 Activity Diagram 模擬使用者互動路徑、撰寫 Project Scope Statement 定義專案範疇。

## 4. 阿公講古導覽系統
* **包含資料**：`MIS-Project-main`網頁原始碼、`期初構想計畫`、`期末構想計畫`、`地圖實作` (系統地圖構想)。
* **專案內容**：應對社區數位轉型需求，設計包含語音導覽與景點說明的觀光網頁，以數位手段保存並推廣地方文化。
* **實作課堂**：資訊管理
* **負責部分**：運用 Business Modle 分析目標使用者需求、繪製 Activity Diagram 模擬使用者互動路徑。
